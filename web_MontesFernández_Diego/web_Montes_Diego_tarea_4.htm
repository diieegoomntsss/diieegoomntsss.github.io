Introduccion

En internet debemos tener comportamiento responsable y proteger nuestra informacion siempre.

Fuentes de calidad

Consejos basicos

  • Proteger tus contrasenas siempre
  • No compartir datos personales en redes inseguras
  • Usar informacion confiable
    • Verificar paginas web antes de usar
    • No difundir rumores ni noticias falsas
  • Respetar a otros usuarios
    1. No insultar ni acosar
    2. Comentar de manera constructiva

Pasos para seguridad online

  1. Crear contrasenas fuertes y cambiarlas regularmente
  2. Comprobar configuraciones de privacidad en redes sociales
  3. Instalar antivirus actualizado
  4. No abrir enlaces sospechosos
